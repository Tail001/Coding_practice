# a011: Kindergarten Counting Game
Content
算一算每行有幾個字（word）。

Word的定義是連續的字元（letter: A~Z a~z）所組成的字。

Input
一段文字(string)
Output
字數(int)
```
Sample Input #1
Hello everybody!!
This is school principal speeking.
Sample Output #1
2
5
```
